Largest Number in an Array in JavaScript

Finding the largest number in an array can be done quickly using JavaScript's built-in methods.

Using JavaScript

Below is an example of a simple function to find the largest number in an array using Math.max(). 


function findLargestNumber(arr) {
    return Math.max(...arr);
}

const numbers = [3, 1, 7, 5];
const result = findLargestNumber(numbers);
console.log(result);

Output: 7.

Explanation

- Math.max(...arr): Finds the largest number in the array by spreading the array elements as individual arguments to Math.max().

This approach is short, easy to understand, and efficient for small arrays.

Conclusion

Using Math.max() with the spread operator is a quick and concise way to find the largest number in an array.
