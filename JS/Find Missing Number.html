Find Missing Number in an Array

If you have a sequence of numbers and one number is missing, you can easily find it using a simple formula in JavaScript.

Using JavaScript

Here's a function that calculates the missing number: 


function findMissingNumber(arr, n) {
    const totalSum = (n * (n + 1)) / 2; 
    const arraySum = arr.reduce((sum, num) => sum + num, 0); 
    return totalSum - arraySum;
}
// Example:
const numbers = [1, 2, 3, 5]; 
const n = 5; 
const result = findMissingNumber(numbers, n);
console.log(result);

Output: 4

How It Works

- The formula (n * (n + 1)) / 2 gives the total sum of the first n numbers.
- The reduce() method calculates the sum of the array elements.
- Subtract the sum of the array from the total sum to find the missing number.

For example, in the array [1, 2, 3, 5]:

  • Total sum of 1 to 5 is 15.
  • Array sum is 11.
  • Missing number is 15 - 11 = 4.

Why This Works

This method is simple, easy to understand, and works efficiently even for large arrays.
