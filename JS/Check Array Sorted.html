Check if an Array is Sorted

To check whether an array is sorted in ascending order, you can use a simple loop to compare each element with the next one.

Simple Example

Below is an example to determine if an array is sorted in ascending order: 


function isArraySorted(arr) 
{
    for (let i = 0; i < arr.length - 1; i++) 
    {
        if (arr[i] > arr[i + 1]) 
        {
            return false; 
        }
    }
    return true;
}
const numbers = [1, 2, 3, 4, 5];
const isSorted = isArraySorted(numbers);
console.log(isSorted);

Output: true

How It Works

- The function loops through the array from the first to the second last element.
- For each element, it checks if it is greater than the next one.
- If such a pair is found, the array is not sorted, and the function returns false.
- If no such pair is found, the function returns true.

Why Use This Method?

This method is simple and efficient, with a time complexity of O(n). It works well for small and medium-sized arrays.

Conclusion

Checking if an array is sorted can be done easily with a loop. This approach is straightforward and effective for basic validation.
