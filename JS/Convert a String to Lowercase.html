How to Convert a String to Lowercase in JS?

In JavaScript, you can easily convert a string to lowercase using the built-in toLowerCase() method. This method returns a new string with all characters converted to lowercase.

1. Using the toLowerCase() Method

The toLowerCase() method is used to convert all the characters in a string to lowercase. It does not modify the original string but returns a new one. 


let str = "HELLO, WORLD!";
let lowerStr = str.toLowerCase();
console.log(lowerStr);  // Output: hello, world!

2. Example Explanation

In the example above, we use toLowerCase() on the string "HELLO, WORLD!". The result is a new string with all characters converted to lowercase.

Output: hello, world!

Conclusion

The toLowerCase() method is a simple and effective way to convert a string to lowercase in JavaScript. It is widely used for various purposes, including text comparison and formatting.
