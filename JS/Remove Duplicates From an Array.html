How to Remove Duplicates From an Array in JavaScript

Removing duplicates from an array can be achieved using various methods in JavaScript. Below are some common approaches to achieve this.

1. Using Set

  • The Set object automatically removes duplicate values when you pass an array to it.
  • You can convert the Set back to an array using the Array.from() method or spread syntax.

Example:

        
let numbers = [1, 2, 2, 3, 4, 4, 5];
let uniqueNumbers = [...new Set(numbers)];
console.log(uniqueNumbers);

Output: [1, 2, 3, 4, 5]

2. Using filter() Method

  • You can use the filter() method to check if an element has already occurred in the array.

Example:

        
let numbers = [1, 2, 2, 3, 4, 4, 5];
let uniqueNumbers = numbers.filter((value, index, array) => 
array.indexOf(value) === index);
console.log(uniqueNumbers);

Output: [1, 2, 3, 4, 5]

3. Using reduce() Method

  • The reduce() method can be used to accumulate unique values into a new array.

Example:

        
let numbers = [1, 2, 2, 3, 4, 4, 5];
let uniqueNumbers = numbers.reduce((acc, value) => {
    if (!acc.includes(value)) acc.push(value);
    return acc;
}, []);
console.log(uniqueNumbers);

Output: [1, 2, 3, 4, 5]

Summary:

Method Approach
Set Automatically removes duplicates and is the simplest method.
filter() Uses a callback function to check if the element already exists.
reduce() Accumulates unique values into a new array.
