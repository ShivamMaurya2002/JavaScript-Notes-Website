Output of console.log(3 + 2 + "7")

The given JavaScript code demonstrates how the + operator behaves when used with numbers and strings.

Code:

        
console.log(3 + 2 + "7");

Output:

57

Explanation:

  • First, 3 + 2 is evaluated, resulting in 5.
  • Then, 5 + "7" is evaluated. Since one operand is a string, the number 5 is converted to a string.
  • Finally, the two strings "5" and "7" are concatenated, resulting in "57".
