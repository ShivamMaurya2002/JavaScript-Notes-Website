Different Ways to Access an HTML Element in JavaScript

JavaScript provides various methods to access and manipulate HTML elements in the DOM (Document Object Model). Here are some common methods:

1. getElementById()

This method is used to access an element by its unique id attribute.

Example:

        


Hello, World!



const element = document.getElementById("exampleId");
console.log(element.textContent); // Output: Hello, World!

2. getElementsByClassName()

This method selects all elements with a specific class name and returns a collection of elements.

Example:

        


First Element


Second Element



const elements = document.getElementsByClassName("exampleClass");
console.log(elements[0].textContent); // Output: First Element

3. getElementsByTagName()

This method selects all elements with a specified tag name and returns a collection.

Example:

        


Paragraph 1


Paragraph 2



const paragraphs = document.getElementsByTagName("p");
console.log(paragraphs[1].textContent); // Output: Paragraph 2

4. querySelector()

This method returns the first element that matches a specified CSS selector.

Example:

        


Example Element



const element = document.querySelector(".example");
console.log(element.textContent); // Output: Example Element

5. querySelectorAll()

This method returns all elements that match a specified CSS selector.

Example:

  • Item 2

const items = document.querySelectorAll("li");
console.log(items[0].textContent); // Output: Item 1

Summary:

Method Description Example
getElementById() Accesses an element by its id. document.getElementById("exampleId")
getElementsByClassName() Accesses elements by their class name. document.getElementsByClassName("exampleClass")
getElementsByTagName() Accesses elements by their tag name. document.getElementsByTagName("p")
querySelector() Accesses the first element matching a CSS selector. document.querySelector(".example")
querySelectorAll() Accesses all elements matching a CSS selector. document.querySelectorAll("li")
