What Is Inheritance in JavaScript?

Inheritance in JavaScript is a mechanism that allows one object to access and use the properties and methods of another object. It enables code reuse and facilitates the creation of hierarchical structures, a common feature in object-oriented programming.

Types of Inheritance in JavaScript

JavaScript supports two primary types of inheritance:

1. Prototypal Inheritance

Prototypal inheritance involves linking objects through their prototypes. Objects can inherit properties and methods from other objects via their prototype chain. 


const animal = {
    eat: function() {
        console.log("Eating...");
    }
};

const dog = {
    bark: function() {
        console.log("Barking...");
    }
};

// Set prototype of dog to animal
Object.setPrototypeOf(dog, animal);

dog.bark(); // Output: Barking...
dog.eat();  // Output: Eating...

Output: Barking..., Eating...

2. Class-Based Inheritance

Class-based inheritance, introduced in ES6, allows developers to use the class and extends keywords to define and extend classes for inheritance. 


class Animal {
    eat() {
        console.log("Eating...");
    }
}

class Dog extends Animal {
    bark() {
        console.log("Barking...");
    }
}

const myDog = new Dog();
myDog.bark(); // Output: Barking...
myDog.eat();  // Output: Eating...

Output: Barking..., Eating...

Key Points to Remember

  • Inheritance allows sharing of properties and methods between objects.
  • Prototypal inheritance uses the [[Prototype]] chain, while class-based inheritance uses the extends keyword.
  • The super keyword enables access to the parent class's properties and methods.

Conclusion

Inheritance in JavaScript is a powerful concept that enhances code reusability and organization. By leveraging prototypal or class-based inheritance, developers can create efficient and maintainable object hierarchies in their applications.
