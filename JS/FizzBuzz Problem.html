FizzBuzz Problem in JavaScript

The FizzBuzz problem is a classic coding challenge. It prints numbers from 1 to a given number with the following rules:
- Replace numbers divisible by 3 with "Fizz".
- Replace numbers divisible by 5 with "Buzz".
- Replace numbers divisible by both 3 and 5 with "FizzBuzz".

Using JavaScript

Below is an example of solving the FizzBuzz problem using a for loop and conditional statements. 


// Declare the variable
const n = 15;

// Loop and apply conditions
for (let i = 1; i <= n; i++) {
    if (i % 3 === 0 && i % 5 === 0) {
        console.log("FizzBuzz");
    } else if (i % 3 === 0) {
        console.log("Fizz");
    } else if (i % 5 === 0) {
        console.log("Buzz");
    } else {
        console.log(i);
    }
}

Output: 1, 2, Fizz, 4, Buzz, Fizz, 7, 8, Fizz, Buzz, 11, Fizz, 13, 14, FizzBuzz

Explanation

- The loop iterates from 1 to the specified number n.
- if (i % 3 === 0 && i % 5 === 0): Checks if the number is divisible by both 3 and 5, printing "FizzBuzz".
- else if (i % 3 === 0): Checks if the number is divisible by 3, printing "Fizz".
- else if (i % 5 === 0): Checks if the number is divisible by 5, printing "Buzz".
- else: Prints the number if none of the above conditions are met.

This approach is straightforward and demonstrates basic programming concepts such as loops and conditionals.

Conclusion

The FizzBuzz problem is a simple yet popular coding challenge that helps improve problem-solving skills and understanding of control structures.
