Find Frequency of Char in a String

Finding the frequency of characters in a string can be done using a simple loop and a JavaScript object to store the counts.

Simple Frequency Counter Example

Below is an example to find the frequency of characters in a string: 


function charFrequency(str) {
    let frequency = {};
    for (let char of str) {
        frequency[char] = (frequency[char] || 0) + 1; 
    }
    return frequency;
}
const text = "hello world";
console.log(charFrequency(text));

Output: { h: 1, e: 1, l: 3, o: 2, w: 1, r: 1, d: 1 }

How It Works

- A loop iterates through each character in the string.
- If the character exists in the frequency object, its count is incremented.
- Otherwise, it is added to the object with an initial count of 1.
- This process continues until all characters are counted.

Why Use This Method?

This method is efficient and easy to understand. It handles spaces and special characters as well, making it versatile.

Conclusion

Using an object and a loop, you can easily find the frequency of characters in any string. This method is short and effective.
