Basics of React JS

React is a library in JavaScript used to build user interfaces. It focuses on creating reusable pieces called *components* to make development easier. Let me explain some of the key concepts in simple terms.

1. What is JSX?

JSX stands for **JavaScript XML**. It looks like HTML, but it’s actually JavaScript. You use it to write the structure of your UI in a readable way. 


<h1>Hello, React!</h1>

This is JSX. It makes it easy to write HTML-like code inside JavaScript.

2. Components in React

Components are like building blocks in React. Think of them as small pieces of a UI that can be reused.

Functional Component

A functional component is a simple function that returns JSX. For example: 


function Greeting() {
    return <h1>Hello, World!</h1>;
}

This is a functional component that shows "Hello, World!"

Class Component

A class component is another way to create components. It uses a class and a special method called render(). 


class Greeting extends React.Component {
    render() {
        return <h1>Hello, World!</h1>;
    }
}

Class components are more advanced but work similarly.

3. What are Props?

Props (short for *properties*) are how you send data from one component to another. They make components customizable. 


function Welcome(props) {
    return <h1>Hello, {props.name}!</h1>;
}

If you pass name="John", this will show: "Hello, John!"

4. What is State?

State is like a component’s memory. It keeps track of things that can change, like a counter. 


class Counter extends React.Component {
    constructor() {
        super();
        this.state = { count: 0 };
    }

    render() {
        return <p>Count: {this.state.count}</p>;
    }
}

The state starts with count = 0. You can update it later.

5. Virtual DOM

The Virtual DOM is like a copy of the real web page. React uses this to quickly figure out changes and update only what’s needed, making the app faster.

React updates the UI efficiently using the Virtual DOM.

6. React Lifecycle

React components go through stages like being added to the page, updated, or removed. These stages are called the lifecycle.

For example: 


componentDidMount() {
    console.log("Component was added to the page.");
}

This runs after the component appears on the screen.

Conclusion

React JS is a powerful library that makes creating user interfaces easier. By understanding concepts like JSX, components, props, state, and the Virtual DOM, you can start building your own applications.
