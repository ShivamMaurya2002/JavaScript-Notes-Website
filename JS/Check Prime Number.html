Check if a Number is Prime in JavaScript

A prime number is a number greater than 1 that has no divisors other than 1 and itself. The following code checks if a given number is prime:

Using JavaScript

Below is an example of solving the prime number check using a for loop and conditional statements. 


// Declare the number
const num = 7;

// Check if the number is prime
let isPrime = true;
for (let i = 2; i < num; i++) {
    if (num % i === 0) {
        isPrime = false;
        break;
    }
}

// Output the result
console.log(isPrime); // Output: true

Output: true

Explanation

- The loop starts from 2 and goes up to one less than the number num.
- if (num % i === 0): Checks if the number is divisible by any number other than 1 and itself, meaning it's not prime.
- If a divisor is found, isPrime is set to false and the loop is exited.
- If no divisors are found, the number is prime, and isPrime remains true.

This approach uses a simple loop to check for factors and determines if a number is prime.
