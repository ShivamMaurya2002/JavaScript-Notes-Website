Interpreter in JavaScript

Interpreter in JavaScript is a program that reads and execute JavaScript code line by line.

In other Word, the browser reads and execute code directly without requiring a separate compilation step.

This is why JavaScript is faster and Dynamic for Web Development.

What is Script in JavaScript?

Script in JavaScript is a set of Instructions or rules which a web browser reads, understands and execute.

Script makes web page dynamic and interactive by responding to users actions,updating contents, handling events, and many more...
