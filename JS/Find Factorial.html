Factorial of a Number in JavaScript

The factorial of a number is the product of all positive integers less than or equal to that number. For example, the factorial of 5 is 5 × 4 × 3 × 2 × 1 = 120.

Using JavaScript

Below is an example of a simple function to calculate the factorial of a number using recursion in JavaScript. 


function factorial(num) {
    if (num === 0 || num === 1) return 1;
    return num * factorial(num - 1);
}

const input = 5;
const result = factorial(input);
console.log(result);

Output: 120.

Explanation

- if (num === 0 || num === 1): Handles the base case where the factorial of 0 or 1 is 1.
- num * factorial(num - 1): Recursively calculates the product of the number and the factorial of the previous number.
- The recursion stops when num is 0 or 1.

This approach is efficient for smaller numbers and easy to understand.

Conclusion

Calculating the factorial of a number is a fundamental exercise in recursion and helps in understanding how functions can call themselves to solve smaller subproblems.
