Difference between == and === in JavaScript

In JavaScript, both == and === are comparison operators, but they differ in how they compare values.

1. == (Equality Operator)

The == operator compares values for equality but performs type coercion. This means it converts the values to the same type before comparing them. 

        
console.log(5 == '5');   // Output: true (string '5' is converted to number 5)

2. === (Strict Equality Operator)

The === operator compares both value and type . It does not perform type coercion, so the values must be of the same type to be considered equal. 

        
console.log(5 === '5');  // Output: false (different types: number vs string)

Summary:

  • == compares values after type conversion (type coercion).
  • === compares both value and type without type conversion.
