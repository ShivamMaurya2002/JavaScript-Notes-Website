Swap Two Variables Without Using a Third Variable

Swapping two variables without using a third variable can be done using arithmetic operations like addition and subtraction or XOR.

Using Arithmetic Operations

Below is an example of swapping two variables using addition and subtraction: 


function swap(a, b) {
    console.log('Before Swap:', 'a =', a, 'b =', b);

    a = a + b; // Step 1: Add both variables
    b = a - b; // Step 2: Subtract b from the sum to get the original a
    a = a - b; // Step 3: Subtract the new b from the sum to get the original b

    console.log('After Swap:', 'a =', a, 'b =', b);
}

// Example:
let x = 5, y = 10;
swap(x, y);

Output: Before Swap: a = 5 b = 10 | After Swap: a = 10 b = 5

Using XOR (Bitwise Operator)

Another method to swap variables without a third variable is using the XOR operator: 


function swapWithXOR(a, b) {
    console.log('Before Swap:', 'a =', a, 'b =', b);

    a = a ^ b; // Step 1: XOR both variables
    b = a ^ b; // Step 2: XOR the result with b to get the original a
    a = a ^ b; // Step 3: XOR the result with the new b to get the original b

    console.log('After Swap:', 'a =', a, 'b =', b);
}

// Example:
let p = 7, q = 15;
swapWithXOR(p, q);

Output: Before Swap: a = 7 b = 15 | After Swap: a = 15 b = 7

How It Works

- **Arithmetic Method**: Uses addition and subtraction to avoid requiring a temporary variable.
- **XOR Method**: Uses the XOR bitwise operation to exchange values without storing them temporarily.
- Both methods ensure the values are swapped efficiently.

Conclusion

Swapping two variables without using a third variable is a great way to optimize memory usage and improve performance. Both arithmetic and XOR methods are effective and widely used.
