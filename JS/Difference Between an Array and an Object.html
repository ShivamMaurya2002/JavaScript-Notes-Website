Difference Between an Array and an Object in JavaScript

In JavaScript, both arrays and objects are used to store collections of data. However, they differ in how they store and access that data.

1. Array

  • An array is a collection of elements stored in an ordered list.
  • Each element in an array is indexed with a numeric key starting from 0.
  • Arrays are typically used to store a list of values (e.g., numbers, strings).

Example:

        
let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[1]);

Output: Banana

2. Object

  • An object is a collection of key-value pairs.
  • The keys can be strings or symbols, and values can be of any type.
  • Objects are typically used to represent structured data (e.g., a person, a car).

Example:

        
let person = { 
    name: "John", 
    age: 30,
    job: "Developer" };
console.log(person.name);

Output: John

Summary:

Aspect Array Object
Data Structure Ordered list Key-value pairs
Access By numeric index By key
Use Case Storing lists of items Representing structured data
Methods Array-specific methods like push, pop Object-specific methods like Object.keys, Object.values
