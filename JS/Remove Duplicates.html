Remove Duplicates from an Array in JavaScript

There are several ways to remove duplicates from an array in JavaScript. Below are some common methods:

1. Using a Set

A Set is a built-in JavaScript object that only stores unique values. By converting an array to a Set, duplicates are automatically removed. 


const numbers = [1, 2, 2, 3, 4, 4, 5];
const unique = [...new Set(numbers)];
console.log(unique);  // [1, 2, 3, 4, 5]

Explanation: The Set constructor removes duplicates and the spread operator (...) converts the Set back into an array.

2. Using filter() and indexOf()

The filter() method can be combined with indexOf() to check if the current element is the first occurrence of that value in the array. 


const unique = numbers.filter((value, index, self) => self.indexOf(value) === index);
console.log(unique);  // [1, 2, 3, 4, 5]

Explanation: indexOf() returns the index of the first occurrence of the element, ensuring only the first occurrence is kept.

3. Using forEach() Loop

You can use the forEach() loop to iterate over the array and manually check if the element already exists in a new array. 


let unique = [];
numbers.forEach(item => {
    if (!unique.includes(item)) unique.push(item);
});
console.log(unique);  // [1, 2, 3, 4, 5]

Explanation: The loop checks each element with includes() to see if it is already in the unique array before adding it.

4. Using for Loop

A traditional for loop can also be used to check for duplicates. This is a more manual approach compared to the other methods. 


let unique = [];
for (let i = 0; i < numbers.length; i++) {
    if (!unique.includes(numbers[i])) unique.push(numbers[i]);
}
console.log(unique);  // [1, 2, 3, 4, 5]

Explanation: Similar to the forEach() method, this approach manually checks for duplicates using includes() and adds only unique elements.

Conclusion

These are some of the most common methods for removing duplicates from an array in JavaScript. Each method has its own advantages:

  • Set: The most concise and efficient way to remove duplicates.
  • filter() and indexOf(): Offers flexibility and is useful for custom filtering conditions.
  • forEach() and for: Provide control over the iteration process, useful for complex logic.
