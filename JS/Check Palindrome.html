Check Palindrome in JavaScript

A palindrome is a word, phrase, or sequence that reads the same backward as forward. For example, "level" is a palindrome.

Using JavaScript

Here is a simple example to check if a string is a palindrome by comparing the original string with its reversed version. 


function isPalindrome(str) {
    const reversed = str.split('').reverse().join('');
    return str === reversed;
}

const input = "level";
const result = isPalindrome(input);
console.log(result);

Output: true.

Explanation

- split(''): Splits the string into an array of characters.
- reverse(): Reverses the order of elements in the array.
- join(''): Joins the array back into a string.
- The === operator compares the original string with its reversed version.

This approach does not account for case or special characters, so "Level" or "L-E-V-E-L" would return false.

Conclusion

This is a simple and direct method for checking palindromes when the input string is plain and does not include case or formatting issues.
