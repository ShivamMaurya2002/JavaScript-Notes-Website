Second Largest Number in an Array in JavaScript

To find the second largest number in an array, we can remove any duplicates, sort the array in descending order, and then select the second element.

Using JavaScript

Below is an example of a simple function to find the second largest number in an array using JavaScript. 


function findSecondLargest(arr) {
    const uniqueArr = [...new Set(arr)];  
    uniqueArr.sort((a, b) => b - a);     
    return uniqueArr[1]; 
}

// Example usage:
const numbers = [10, 5, 8, 12, 20, 20];
const result = findSecondLargest(numbers);
console.log(result);

Output: 12

Explanation

- const uniqueArr = [...new Set(arr)]: Removes duplicates from the array using the Set object.
- uniqueArr.sort((a, b) => b - a): Sorts the array in descending order.
- return uniqueArr[1]: Returns the second element in the sorted array, which is the second largest unique number.

This approach works well even if the array contains duplicates and ensures that the second largest unique number is returned.

Conclusion

Using this method, we can efficiently find the second largest number in an array by removing duplicates and sorting. This method has a time complexity of O(n log n) due to the sorting step.
