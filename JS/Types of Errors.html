Types of Errors in JavaScript

In JavaScript, errors are issues that occur when the code is executed. These errors can prevent your program from working properly. Let's look at the main types of errors and how they occur.

1. Syntax Error

A syntax error happens when there's a problem in the structure of your code. It usually occurs if you write code that JavaScript cannot understand. 


// Missing closing parenthesis
console.log("Hello World"; // Syntax Error

Output: Uncaught SyntaxError: missing ) after argument list

2. Reference Error

A reference error happens when you try to use a variable or function that hasn't been defined. 


// Trying to access an undefined variable
console.log(x); // Reference Error

Output: Uncaught ReferenceError: x is not defined

3. Type Error

A type error occurs when a value is not of the expected type. For example, trying to call a number like a function. 


// Calling a number as if it's a function
let num = 42;
num(); // Type Error

Output: Uncaught TypeError: num is not a function

4. Range Error

A range error happens when a value is outside the allowed range. For instance, creating an array with an invalid length. 


// Invalid array length
let arr = new Array(-5); // Range Error

Output: Uncaught RangeError: Invalid array length

5. Eval Error

Eval errors occur when the eval() function is used incorrectly. Note: This error is rare because modern JavaScript doesnâ€™t throw it often. 


// Incorrect usage of eval (rarely encountered)
eval("alert('Hello)"); // Eval Error

Output: Uncaught SyntaxError: Unexpected end of input

6. URI Error

URI errors occur when there's a problem with encodeURI() or decodeURIComponent(). For example, using an invalid character in a URI function. 


// Decoding an invalid URI
decodeURIComponent("%"); // URI Error

Output: Uncaught URIError: URI malformed

Conclusion

Understanding the different types of errors in JavaScript can help you debug your code more effectively. Always check your syntax, variable names, and data types to avoid these errors!
