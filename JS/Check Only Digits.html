Check if a String Contains Only Digits in JavaScript

To check if a string contains only digits, we can use a regular expression to match the string against a pattern that allows only numeric characters.

Using JavaScript

Below is an example of a function to check if a string contains only digits using a regular expression. 


function isOnlyDigits(str) {
    return /^\d+$/.test(str);  
}

// Example usage:
const testString = "12345a";
const result = isOnlyDigits(testString);
console.log(result);

Output: false

Explanation

- /^\d+$/: This regular expression matches a string that consists entirely of one or more digits.
- .test(str): The test() method checks if the string matches the regular expression pattern.
- The string `"12345a"` contains a non-digit character, so the result is false.

This method ensures that the string contains only numeric characters.

Conclusion

Using a regular expression is an efficient way to validate if a string contains only digits. It checks the string character by character and ensures no non-digit characters are present.
